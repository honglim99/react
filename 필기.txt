변수 종류
기본형변수 : 숫자형, boolean형, 문자형
참조형 변수 : 문자열, 배열, 객체

할당 연산자에 의해 데이터가 저장될때
기본형 변수에는 값이 저장된다.
참조형 변수에는 주소가 저장된다.

const a = 3; //a 변수 기본형이므로 3이라는 값이 저장됨
const arr=[1,2,3] // arr 참조형 변수 arr에는 주소가 저장됨
어떤 주소가 저장되는가? [1,2,3]배열을 저장한 주소가 100이라 할때
arr에는 100이 저장된다.

=, ...에 의해서 배열이나 객체를 복사를 할 때 주의해야하는 사항

const arr = [1,2,3]
const arr2 = arr 주소복사 arr2 = 100
const arr3 = [...arr] arr3 = [...arr]배열이 저장된 주소가 저장됨.

//할당연산자에 의해서 배열이 복사되는 경우에는 주소가 복사되므로, 원본데이터가 편집되므로
//주의한다.
arr[1] = 4; 수정한 후,
arr2[1] 출력하면 -> 4 //배열내용이 공유되므로...
//스프레드 연산자에 의한 복사인 새로운 배열이 만들어지면서 복사가 이루어진다.
// 원본 배열이 수정되어도 복사본에 영향을 미치지 않는다.
